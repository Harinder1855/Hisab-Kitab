# Hisab Kitab - Digital Cashbook App

**Hisab Kitab** is a user-friendly mobile application designed to simplify daily expense and income tracking. Built with **Flutter**, it serves as a secure digital cashbook for individuals and small business owners to manage their finances efficiently.

## 🚀 Features

* **Record Transactions:** Quickly add income and expense entries with category, date, and amount.
* **Real-time Balance:** Instantly view your current balance based on recorded transactions.
* **Category Management:** Create and manage custom categories for better organization.
* **Detailed Reports:** Generate daily, weekly, or monthly reports to understand spending patterns.
* **Firebase Integration:** Securely sync data across devices (if applicable) and authenticate users.
* **Intuitive UI:** Clean and simple interface for effortless financial management.

## 🛠️ Tech Stack

* **Frontend:** Flutter (Dart)
* **Backend:** Firebase (Firestore, Auth)
* **State Management:** Provider
